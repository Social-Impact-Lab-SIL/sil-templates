# SIL Repository Templates

This repository contains standard templates, folder structures, and coding guidelines for all Social Impact Lab (SIL) projects.

## How to Use These Templates

When starting a new project, copy the relevant template folder into your new repository and fill in the bracketed fields.

## Templates Included

- `README-template.md` — Standard README for any SIL project repo
- `README-replication.md` — README template specifically for replication packages
- `.gitignore-stata` — Standard .gitignore for Stata-based projects
- `.gitignore-r` — Standard .gitignore for R-based projects
- `.gitignore-python` — Standard .gitignore for Python-based projects
